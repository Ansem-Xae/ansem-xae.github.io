ATA Express

Description:

ATA Express is an SDIO to IDE (PATA) emulator. Currently, it is STRICTLY a hard disk emulator, but it can and will be expanded to CD-ROM/DVD emulation eventually.

You can purchase it from Phenom Mod Here, otherwise, if you are in the EU, you can purchase from Here.

Features:

  • Up to 2TB microSD
  • LBA28 support
  • LBA48 support
  • CHS mode
  • UDMA6 capable
  • Master/Slave select for IDE interface

What’s so special about this product compared to current solutions on the market? Nearly every all SD to IDE adapters on the market use the FC1307A chipset. It comes with a buggy firmware, limited to LBA28 (128GB max) compatibility. This doesn't and is upgradable.

Some use cases include:
  • Finally max out a PS2 70K. Most mods use the SD to IDE adapters with the 128GB limit
  • Portable OG Xbox
  • Retain the disc drive in Dreamcast
  • Retro PC’s, with CF becoming a rare storage solution, this is another solid state option
